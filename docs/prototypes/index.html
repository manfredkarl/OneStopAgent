Design Reference

Design System
• Color: Microsoft Blue #0078D4, Grey #F3F2F1 / #E1DFDD
• Typography: Segoe UI with system fallback stack
• Radius: 4–8 px, subtle elevation shadows
• Animations: Pulse (working agents), smooth transitions, hover lift
• Accessibility: ARIA labels, focus rings, proper contrast ratios
